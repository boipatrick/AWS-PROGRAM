#### Prompt techniques

- Zero-shot prompting
-Few-shot prompting
- Chain f thought prompting
- Self-consistency
- Tree of thoughts(ToT)
- Retrieval Augmented Generation (RAG)
- Automatic Reasoning and Tool-use(ART)
- ReAct prompting


#### RAG
is a naturallanguage processing technique that combines the capabilities of retrieval systems and generative language models to produce high-quality and informative text outputs.

It incorporates two components:
1. A retrieval system - retrieves relevant information from a large corpusof text data, such as a knowledge base, web pages or other textual sources

2. A generative language model - E.g GPT-3 that can generate natural language text.

### RAG business applications
Building intelligent question-answering systems
Expanding and enriching existing knowledge bases
Generating high quality content

### Fine-Tuning
this is another way to improve the performance of a foundational model even further. It refers to the process of taking a pre-trained language model and further training it on a specific task or domain-specific dataset. 
Fine tuning the FM base model can improve performance

There are two ways to fine tune a model:
1. Instruction fine-tuning uses examples of how the model should respond to a specific instruction. Prompt tuning is a type of instruction fine-tuning

2. Reinforcement Learning from human feedback provides human feedback data, resulting in a model that is better aligned with human preferences
